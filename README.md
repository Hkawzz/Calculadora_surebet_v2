# Calculadora_surebet_v2
-É uma calculadora feita no django, tendo 2 opções para interagir
-Usada pra verificar se, apostar em diferentes possibilidades, garantem ganhos, independete do resultado final

# Como usar

1. Clone o repositorio do github
2. No terminal de o comando (python manage.py runserver)
3. Abra o link indicado (localhost)
